# runtime/

This directory holds regenerable output for the `scitex-notification` package — logs,
cache, PID files, and any other data that can be re-created from configuration +
source.

Everything under `runtime/` is gitignored. See
[local-state-directories.md](https://github.com/ywatanabe1989/scitex-dev/blob/main/src/scitex_dev/_skills/general/01_ecosystem/06_local-state-directories.md)
for the canonical layout.
